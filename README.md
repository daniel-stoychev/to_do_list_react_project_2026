# Task Manager App (React Web)

A simple **mobile‑first Task Manager web application**.
This project is built as a **junior‑friendly portfolio project**, focusing on clean React fundamentals, clear state management, and responsive UI.

---

## 🎯 Project Goals

- Practice **React fundamentals** (hooks, state, components)
- Build a **real, usable app** from scratch
- Apply **mobile‑first responsive design**
- Create a solid **portfolio project** with clear scope

---

## 🧩 Features (MVP)

- Add Task field
- Button for Adding a task
- Button for Deleting a task
- Button for Marking a taks as completed
- Button for Terminail the whole list with cinfirm functionality
- Keep tasks state in LocalStorage
- Simple & celan UI

---

## 🚀 Tech Stack

- **React** (Web)
- **Vite**
- **Tailwind CSS**
- CSS (CSS Modules)
- localStorage
